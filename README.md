# chat_serve
聊天服务后端
- 本项目为即时聊天服务的后端，由Springboot+vue3+mysql+websocket实现，功能参考微信等聊天软件设计

- 技术栈 
  - 后端 
    - 选用编程语言 Java
    - web框架SpringBoot
    - db MySQL 持久存储
    - nosql 缓存 Redis
    - 全双工通信框架 WebSocket
  - 前端 
    - 前端框架Vue3
    - Typescript
    - UI样式 Css、ElementPlus
    - 网页路由 vue-router
    - 全双工通信框架Websocket
功能概况
- 登录注册模块
  
- 加好友
  
- 好友搜索与列表
  
- 发起群聊
  
- 消息单聊与群聊
  
- 群成员列表等模块
